# Moneyraham Homepage Design

`Moneyraham_homepage_Disign`은 한국어 우선 홈페이지를 설계하고 구현하기 위한 Codex 스킬입니다. 반응형 화면, 접근성, CTA, 실제 미리보기 검증과 작업 경험 기록을 하나의 흐름으로 다룹니다.

## Codex 설치

Codex가 설치된 컴퓨터의 터미널에서 실행합니다.

```powershell
python "$env:USERPROFILE\.codex\skills\.system\skill-installer\scripts\install-skill-from-github.py" `
  --repo uncheon0912/bujasaram-homepage-design `
  --path skills/moneyraham-homepage-design
```

설치가 끝나면 Codex 새 작업을 열고 아래처럼 요청합니다.

```text
$moneyraham-homepage-design을 사용해서 한국어 반응형 홈페이지를 만들어줘.
```

또는 홈페이지 제작을 자연어로 요청하면 자동으로 적용될 수 있습니다.

## 포함 범위

- 한국어 우선 홈페이지 설계·구현 워크플로
- 브랜드, 방문자 목표, CTA 중심의 페이지 구조
- 반응형·접근성·상호작용·미리보기 검증 기준
- 검증된 경험을 다음 작업에 재사용하는 기록 방식
