## Package boundary

Publish this folder as the personal skill. Do not copy the project's `.codex/skills/ui-ux-pro-max` folder into it unless the published repository intentionally includes a derivative copy.

## Upstream material

The installed `ui-ux-pro-max` source is MIT licensed. If its code, text, templates, data, or substantial portions are copied or modified for the student package, include the original copyright notice and MIT license in the published repository.

## Student-ready release check

- Confirm the skill name is `moneyraham-homepage-design` and the display name is `Moneyraham_homepage_Disign`.
- Remove project-specific assets, client copy, credentials, and local absolute paths.
- Include only required skill files and useful references.
